# tahara-kouichi

## プロフィール

**名前：**田原 広一

1985年生まれ。石川県出身高校卒業後、東京IT会計・資格の学校TACにて学びファイナンシャルプランナーの資格を獲得。資格の学校TACにて税理士講座の講師を務め、年間200回以上の講義を行う。その傍ら経理代行、融資支援、相続メディア運用などの取り組みを開始。29歳に税理士有資格者となり、SoLaboを設立。

## 経歴

### 毎月20本の記事で運命を変えた男

田原社長は、専門学校に通いながら資格の学校TACで学び、その後、税理士講座の講師を務める傍ら大学院にも通い....と、がむしゃらな努力を重ね20代で税理士資格を取得しました。そんな田原社長が「楽しそうだから」と挑戦したのが、新しいメディア「相続情報ラボ」での記事企画・執筆の仕事でした。

相続の知識もなければ、SEOの知識、メディア運営の経験もなかった田原社長は、毎月20本の記事を粘り強く執筆し、ノウハウを磨き上げました。その結果メディアはなんと、月間100万PVを達成！この成果が高く評価され、子会社の代表に抜擢されました。これが株式会社SoLaboの始まりです。

### 明るく元気で前向きなヤツ大歓迎

「知識がゼロの相続メディアで成功できたなら、知識のある融資ならもっと勝てる！」と確信し、融資メディアをスタートしました。すると狙い通り大ヒット！月200件以上の問い合わせがくるようになり、毎月5～10件の融資支援を行うようになりました。

そこで得たSEOの知識を武器に、SEOコンサル事業にも挑戦。SEOコンサルではビッグワードで1位を獲得し、さらなる成長を遂げました。勢いそのままにオフィスを市ヶ谷へ移転し、採用活動も進めました。

採用基準は、「明るく元気で前向きなヤツ大歓迎！」。この基準を満たす人を採用できたので、社員同士が信頼し合い、温かな雰囲気の中で成長し続ける会社へと進化していきました。

### 急成長も怖くない

田原社長の行動力は止まりません。融資コンサル事業とSEOを中心とするデジタルマーケティング事業でも成功しました。融資コンサル事業では月500件の問い合わせを獲得します。さらに、デジタルマーケティング事業部は各分野の専門家が集まり、一気に支援の幅を広げました。その結果、毎年2倍の成長を遂げ、2022年には「アジア太平洋地域の急成長企業ランキング」に選出。特にマネジメントコンサルティング部門ではアジア太平洋地域で3位の成長率を記録しました｡

普通なら、急成長が怖いと感じることもあると思います。そんな急成長をものともせず、常に「ワクワク」を感じる方向へ突き進んでいける田原社長。その背景にあるのは、「最悪の場合はメディアを売ればいい！」というリスクヘッジの考え方。最悪の事態を想定しつつ、未来への投資を惜しまない経営スタイルが、さらなる成長を切り拓きました。

### 立ちはだかる上場という壁

創業以来、5期連続でほぼ2倍の成長を遂げ、急成長企業ランキングにも選出されたSoLabo。「あれ、上場いけるんじゃね？」と田原社長が思い立ち、上場準備がスタート！成長を支えた幹部が上場準備に時間を費やすようになりました。

しかし、その裏で、事業に割ける時間が激減。さらに、コロナ禍の影響で融資コンサル事業の売上は下がってしまいます。その結果、SoLaboは初めて減収・赤字という試練に直面しました。急成長の先に待っていたのは、挑戦と挫折の両方が入り混じる、次のステージへの幕開けでした。

### 再スタートから、新たな挑戦へ

「ビジネスモデル的にも、今すぐ上場できる実力はない。愚直に営業をして伸ばしていこう！」と田原社長は決意し、再スタートを切りました。その結果、融資コンサル事業は回復し、数年前に立ち上げた補助金事業も軌道に乗りました。さらに、デジタルマーケティング事業も好調。事業の3本柱がしっかりと築かれました。

このタイミングで従業員数は100名を突破。勢いそのままに、田原社長は新たな挑戦を次々と仕掛けます。自治体支援プロジェクト、若手税理士・会計士のためのコミュニティ「ふらっと」の創設、そしてBPO事業の立ち上げなど、多岐にわたる事業を展開しました。

### ワクワクしながら突っ走る

今まで、様々な事業を手掛け社会課題を解決することを大切に感じるようになった田原社長。企業を成長させ喜んでもらうことが、ワクワクの源泉になりました。都市部近郊の企業だけでなく、自治体、地域事業者、金融機関との連携を強化し、社会課題解決の幅広さや解決への奥深さを養っていく。これからも田原社長の想いに共感してくれる人たちを集め、ワクワクしながら一致団結して突っ走ります。
